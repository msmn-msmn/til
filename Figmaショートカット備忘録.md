# ★Figmaのショートカット
## 🐻‍❄️概要
Figmaを使うときのショートカットの備忘録

## ❇️ショートカット一覧
| ショートカットキー | 内容 |
|--------|---------------------------------------------------------|
| V　| オブジェクトの移動　|
| K | 拡大・縮小 |
| F | フレーム（枠組み作成） |
| Shift + S | フレームを視覚的にグルーピング |
| R | シェイプツール　長方形 |
| L | シェイプツール　直線 |
| O | シェイプツール　楕円 |
| P | シェイプのパス調整・変形 |
| T | テキストツール 　クリック挿入で自動幅　<br> 　　　　　　　　ドラッグ挿入で固定幅　|
| C | コメントツール　キャンパス上にコメントを残せる |
| Shift + A | オートレイアウトの追加 |
| Alt + Shift + A | オートレイアウトの削除 |
| Ctrl + Alt + Shift + A | 全体をオートレイアウトに追加 |
| Ctrl + Z | 一個前の操作に戻す |
| Ctrl + Shift + Z | 戻しすぎた操作を進める |
